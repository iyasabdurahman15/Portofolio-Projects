# Nether E-Commerce Marketing Analysis
## Overview
Proyek ini bertujuan membuat dashboard interaktif mengenai analisa marketing pada penjualan di perusahaan e-commerce virtual.  Data yang digunakan dikoneksikan dari local postgresql database dengan mode penyimpanan kombinasi dari directqquery dan import. Mode directquery untuk data order dan return, sedangkan mode import untuk data dimensi yang biasanya digunakan untuk filtering atau slicer.
