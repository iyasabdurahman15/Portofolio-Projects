# Loan-Performance-Dashboard-Project
# Overview:
Proyek ini dibuat sebagai final proyek bootcamp data analytics with Excel Human Initiative Academy X dibimbing.id.
Proyek ini bertujuan untuk membuat dashboard interaktif yang memvisualisasikan performa pinjaman berdasarkan beberapa metrik
seperti jenis produk, jenis transaksi, plafon pinjaman, demografi peminjam, tingkat kegagalan pembayaran dll.

# Key Insights and Visualizations:
- Portofolio pinjaman terdiri dari 279 kontrak dengan total plafon mencapai Rp 24.636.300.000 dan menghasilkan total bunga sebesar Rp 15.394.291.026
- Kota Makassar memiliki proporsi debitur dengan resiko kredit "Good" yang paling tinggi, sementara Kota Palembang memiliki proporsi debitur dengan resiko "Good" paling rendah
- Kota Jambi dan Makassar memiliki jumlah plafon dan bunga pinjaman paling tinggi yang mengindikasikan potensi pertumbuhan ekonomi karena permintaan kredit yang besar
