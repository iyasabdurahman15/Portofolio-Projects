# Console Games Dashboard
## Overview
Proyek ini bertujuan untuk membuat dashboard interaktif game dan platform game tersebut. Dashboard ini memiliki dua halaman dimana halaman pertama mengenai game seperti total game dari seluruh platform, total game berdasarkan genre, sedangkan itu halaman kedua mengenai game platform seperti penjualan pc atau playstation per tahunnya, produk yang diskontinu per tahunnya dll.
