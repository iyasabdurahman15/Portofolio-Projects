# Olist Transaction Dashboard
## Overview
Proyek individu ini bertujuan membuat dashboard intuitive mengenai penjualan/sales pada perusahaan e-commerce virtual. Data pada dashboard ini diambil dari postgresql database dengan mode penyimpanan import. Terdapat dua halaman pada dashboard ini dimana halaman pertama berisi report overview//keseluruhan mengenai penjualan sementara halaman kedua berisi report mengenai produk seperti penjualan berdasarkan kategori, top produk dll.
