# Customer Complaints Dashboard
## Overview
Proyek individu ini bertujuan membuat dashboard interaktif mengenai komplain pembeli pada perusahaan yang ada di Amerika Serikat pada periode 2013 - 2016. Dataset ini diimpor dari website, lalu di import ke dalam local postgresql database. Setelah itu, dikoneksikan ke power bi dari local database dengan mode penyimpanan directquery. "Customer Complaints Report.pbix" adalah file berisi model dan visual dashboard dari proyek ini
